# Análisis del estrés académico en estudiantes universitarios de Bangladesh mediante Python en el 2025

## Descripción

Este proyecto realiza un análisis exploratorio de datos sobre el estrés académico en estudiantes universitarios de Bangladesh, utilizando Python como herramienta principal para el procesamiento, análisis y visualización de la información.

El objetivo es identificar patrones relacionados con distintos factores que pueden influir en el nivel de estrés académico, como variables personales, académicas y sociales presentes en el conjunto de datos.

## Objetivo general

Analizar el comportamiento del estrés académico en estudiantes universitarios de Bangladesh mediante técnicas de análisis de datos en Python.

## Objetivos específicos

- Cargar y explorar el conjunto de datos.
- Preparar los datos para su análisis.
- Identificar variables relacionadas con el estrés académico.
- Generar visualizaciones que faciliten la interpretación de los resultados.
- Aplicar funciones en Python para organizar el análisis por tipos de estrés.

## Tecnologías utilizadas

- Python
- Pandas
- Matplotlib
